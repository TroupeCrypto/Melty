## Adding a New Project

When merging a new repository into the Troupe Inc. universe:

1. Create a new directory under the appropriate category
2. Preserve git history using subtree merge or similar techniques
3. Update this document with project details
4. Add project-specific documentation
5. Update main README.md if needed

## Project Categories

Projects are organized by type:
- **apps/** - Standalone applications
- **libs/** - Shared libraries and utilities
- **services/** - Backend services and APIs
- **tools/** - Development and deployment tools
- **docs/** - Documentation and resources
